Python

This chapter will demonstrate installing the Hindsight python package with the Behave module, and using it from the command line or within your Python scripts

Installing the Hindsight package

The package is hosted on PyPI at http://pypi.python.org/pypi/hindsight and is available through pip or by direct download. Assuming you have Python and pip installed on your system, run 

pip install hindsight

and you will have installed the behave python client and its dependencies. Depending on your system, you may have to prefix the command with "sudo"

Using from the command line

Installing the package will have placed a binary on your path named "behave-cli". The output of "behave-cli --help" is as follows: 

behave-cli HOST KEY [-u USERNAME] [-p PASSWORD] [-d DIR] [-m]

This would download all .feature files from the PROJ project at "http://localhost:2990/jira" to the features directory (relative to the directory behave was run from), including all scenarios tagged as manual.

Host and Key are the only required arguments, which defaults to an anonymous login downloading all features to the directory the command is run from.

Using within a script

You can use the Jira Connector within your own scripts to download features from Behave 

#!/usr/bin/env python

from hindsight.behave import JiraConnector

jc = JiraConnector()
jc.fetch(
	host='http://localhost:2990',
	username='behaver',
	password='atdd4me',
	key='PROJ',
	dir='features',
	manual=True
)

This will download all features from PROJ at http://localhost:2990 into the features directory, including manual tagged scenarios
