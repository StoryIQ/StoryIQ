Enabling or Disabling Behave on Projects

You can opt to have Behave enabled by default on all projects, or you can pick and choose which projects to enable them on. In the administration area underneath the plugin section, select "Enabled Projects".

If you have all projects enabled by default, then any new project will be enabled on creation. Otherwise, new projects will be disabled by default.
