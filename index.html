JIRA BEHAVE

Make code behave itself.

Write acceptance tests to know when you've got the code you want. Get a heads up when things break. Keep on top of your software project.

Test Automation

Don't let your Test engineers become expensive keyboard automation tools, JIRA Behave can generate acceptance tests for the leading open source BDD automation framework, Cucumber. Cucumber supports a number of programming languages including: Java, Ruby, and Scala.

To automate or not to automate?

Test automation is not suited to every situation but JIRA Behave has it covered. A mixture of automated and manual testing can be achieved by marking individual acceptance criteria as "Manual" to stop them being run by Cucumber.

Cucumber Report Plugin for Bamboo

The Atlassian Cucumber integration doesn't stop at JIRA, Bamboo users can take advantage of the Cucumber Report Plugin. The plugin is available for download on the Atlassian Marketplace
