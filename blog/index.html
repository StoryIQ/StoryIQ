Security testing with Selenium and the Zed Attack Proxy (ZAP)

A few weeks ago I had the pleasure of speaking at the 2012 Selenium Conf. and the Video of my talk has now been published.

Recreating my demo

  1. Install Chrome if not already installed
  2. Start Jenkins on port 80. I used a fresh install of Jenkins with no configuration changes
  3. Start the Zed Attack Proxy (ZAP), no additional configuration required.
  4. Run the JUnit test "JobManagementTest" in my GitHub project. Note: this test uses Chrome
